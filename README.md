## Description

CRUD using Prisma, NestJS and Postgres

## Installation

```bash
$ npm install
```

## Prisma
$ npx prisma init
$ npx prisma migrate dev --name init


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
